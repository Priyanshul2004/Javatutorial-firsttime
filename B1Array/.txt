2. Find the Missing Number
Problem: Given an array containing n distinct numbers taken from 0, 1, 2, ..., n, find the one number that is missing.
Difficulty: Easy
Concepts: Summation formula, XOR
Variants: Find two missing numbers, or find the missing number in an unsorted list.

3. Maximum Subarray Sum (Kadane's Algorithm)
Problem: Find the contiguous subarray (containing at least one number) with the largest sum and return its sum.
Difficulty: Medium
Concepts: Dynamic Programming
Common Follow-up: Can you modify the solution to return the subarray itself?

5. Merge Intervals
Problem: Given an array of intervals where intervals[i] = [start, end], merge all overlapping intervals and return an array of the non-overlapping intervals.
Difficulty: Medium
Concepts: Sorting, Merging Intervals
Common Follow-up: Solve it with in-place modifications.




6. Best Time to Buy and Sell Stock
Problem: Given an array prices where prices[i] is the price of a given stock on the ith day, find the maximum profit you can achieve by buying and selling one share of the stock.
Difficulty: Easy to Medium
Concepts: Dynamic Programming
Variants: Maximum profit with multiple transactions allowed, or with a cooldown period.

Problem: Rotate an array nums of n elements to the right by k steps.
Difficulty: Medium
Concepts: Array Reversal, In-place Manipulation
Variants: Rotate to the left or rotate a 2D array.
